## 二层发现（数据链路层）

### ARP协议扫描特点：

* ### 优点：

  扫描速度快，可靠性高。

* ### 缺点：

  不可路由，只能发现本网段内的主机。

## 扫描工具arping的使用：

* ### 基本语法：


* ###常用命令:  

  ```bash
  #扫描某个IP主机是否存活
  arping 192.168.5.174 -c 1  
  
  #检测网段内是否有arp欺骗
  arping 192.168.1.1 -d
  ```

* ### 脚本编写：
  > ##### 1.扫描网段内所有存活的主机：
  
  ```bash
  dfgdfsg 
  ```  
  > ##### 2.扫描网段内自定义的IP地址：
  
  ```bash
  dfgdfsg
  ```
